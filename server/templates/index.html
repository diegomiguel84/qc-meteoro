A Python weather service

Available services

  • get_cities: Returns a list of cities (a list of dictionaries with keys: code, name)
    • contains: optional - the cities returned should contain the given letters
  • get_forecast: Returns the forecast for a city (a list of dictionaries with keys: date, temperature, humidity, uv, chill, status)
    • city: mandatory - the name of the city for which the forecast is required
  • change_provider: Changes the forecast provider of the app
    • name: mandatory - the name of the forecast provider to use (current implementations are: yahoo, mock
